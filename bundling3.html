Penyelesaian Sistem Persamaan Linear 3 Variabel

Eliminasi dan Substitusi untuk mencari nilai x, y, dan z

1. Model Matematika

x = harga 1 Tas Tutup Botol

y = harga 1 Bingkai Kaca dari CD

z = harga 1 Lampu Nanas

Persamaan:

1) 2x + y + 3z = 270.000

2) x + y + 2z = 190.000

3) 2x + 2y + z = 245.000

2. Eliminasi Persamaan (1) dan (2)

(2x + y + 3z) − (x + y + 2z) = 270.000 − 190.000

x + z = 80.000

→ Persamaan (4)

3. Eliminasi Persamaan (2) dan (3)

Kalikan persamaan (2) dengan 2:

2x + 2y + 4z = 380.000

Kurangkan dengan persamaan (3):

(2x + 2y + 4z) − (2x + 2y + z) = 380.000 − 245.000

3z = 135.000 → z = 45.000

4. Substitusi z ke Persamaan (4)

x + z = 80.000

x + 45.000 = 80.000

x = 35.000

5. Substitusi x dan z ke Persamaan (2)

35.000 + y + 90.000 = 190.000

y + 125.000 = 190.000

y = 65.000

✔ Hasil Akhir

(x, y, z) = (35.000, 65.000, 45.000)
